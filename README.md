# 🚌💨 Carrera de Buses

¡Bienvenido a la carrera más veloz entre dos titanes del transporte público!  
El 🟥 **Bus Rojo** y el 🟦 **Bus Azul** están listos para demostrar quién es el rey de la ruta.

---

## 🎮 Controles

- ▶️ **Iniciar Carrera**: ¡Que empiece la acción!
- 🔄 **Reiniciar**: Restablece los buses a la línea de salida.

---

## 🏁 Progreso de la carrera

**Los buses avanzarán con cada clic en el botón "Iniciar Carrera". ¡El primero en llegar al final gana!** 🏆

---

## 🧩 Cómo funciona

Este proyecto está construido con:
- HTML + CSS para la estructura y estilos 🖌️
- JavaScript para la lógica de la carrera ⚙️

---

## 📂 Archivos Clave

- `index.html` → Estructura base del juego.
- `style.css` → Estilos para la pista y los buses.
- `main.js` → Lógica de movimiento y control de botones.

---

¡Prepárate para gritar "¡Vamos, bus azul!" o "¡Corre, bus rojo!" mientras se decide el campeón de la ruta! 🚦🎉
